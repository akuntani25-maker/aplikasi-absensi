# UI — React Native + NativeWind + Reanimated
## NativeWind v4: Tailwind syntax di RN → `className="bg-blue-500 p-4 rounded-lg"`
## Reanimated 3: `useAnimatedStyle`, `withSpring`, `withTiming` untuk 60fps animations
## Gesture Handler: untuk swipe, pan, pinch gestures
## FlashList (Shopify): pengganti FlatList yang lebih performant
## expo-image: pengganti Image dengan caching + placeholder
## Platform-specific: `Platform.select({ ios: ..., android: ... })` atau `.ios.tsx`

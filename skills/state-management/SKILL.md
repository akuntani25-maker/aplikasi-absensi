# State — Zustand + TanStack Query
## Server state: TanStack Query (caching, refetching, offline)
## Client state: Zustand (lightweight, no boilerplate)
## Persistent state: MMKV (fast) atau expo-secure-store (sensitive)
## Offline: TanStack Query `networkMode: 'offlineFirst'` + MMKV persistence
